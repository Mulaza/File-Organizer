# File-Organizer
A script that scans throught the current directory and organizes all loose files into neat folders by file extension

## Dependencies
[colorama](https://pypi.org/project/colorama/)

## Installation
```python
pip install organizer-cli
```

## Commands

* safe
* all
* images
* audios
* videos
* office
* vector
* gif
* photoshop
* pdf
* font
* code
* safe
* all
